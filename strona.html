Format CSV (comma-separated values)jest to format tekstowy, którego dane rozdzielone są znakiem przecinka (,). Taki format pozwala na odczytanie plików poprzez aplikację arkuszy kalkulacyjnych dzięki czemu umieszcza on dane do odpowiednich kolumn i wierszy. Format JSON (JavaScript Object Notation) jest to format tekstowy najczęściej stosowany wymiany danych pomiędzy maszynami połączonymi siecią. Dane zakodowane w tym formacie zwierają stosunkowo mało znaków co sprawia, że charakteryzuje się on wysoką czytelnością. Format XML (Extensible Markup Language) jest to format tekstowy, dzięki któremu możemy przechowywać, przesyłać i zmieniać dowolne dane. Umożliwia on łatwą wymianę pomiędzy różnymi systemami i jest łatwy od odczytu zarówno dla użytkownika jak i maszyny.
